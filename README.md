# PDF Editor 📝

A simple open-source PDF editor built with Python, Tkinter, and pikepdf.  
Features:
- Reverse PDF
- Delete Pages
- Replace Pages
- Rotate PDF
- Encrypt PDF
- Extract Pages
- Extract Images
- Compress PDF

## Installation
```bash
git clone https://github.com/yourusername/pdf-editor.git
cd pdf-editor
pip install -r requirements.txt
